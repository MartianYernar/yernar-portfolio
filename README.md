# Yernar Mars — portfolio site

A single-page static portfolio. No build step — just HTML + CSS.

## Run locally
Open `index.html` in a browser (or `python -m http.server` then visit
`http://localhost:8000`).

## Add your images (later)
Drop image files into an `images/` folder with these exact names — they'll
appear automatically (until then, clean gradient placeholders show):

| File | Project |
|------|---------|
| `images/research.jpg` | Minimum-dose PH screening |
| `images/smartgarbage.jpg` | SmartGarbage bin |
| `images/soilyget.jpg` | SoilyGet robot |
| `images/automation.jpg` | AI automation for clinics |
| `images/mentoria.jpg` | Mentoria academy |

Use landscape images (~16:9). A short demo `.mp4` can be linked from the cards
or hosted on YouTube and embedded later.

## Deploy (GitHub Pages)
Already configured if pushed to GitHub with Pages enabled on `main` → site at
`https://martianyernar.github.io/yernar-portfolio/`.
